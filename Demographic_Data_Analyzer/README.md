# Demographic Data Analyzer

This project analyzes demographic data from the **Adult Census dataset** using Python and Pandas.

## What This Project Does

The program reads `adult.data.csv` and calculates several statistics related to education, salary, working hours, and demographics.

## Analysis Performed

* Counts the **number of people in each race**.
* Calculates the **average age of men**.
* Finds the **percentage of people with a Bachelor's degree**.
* Computes the **percentage of people with advanced education (Bachelors, Masters, Doctorate) earning more than 50K**.
* Computes the **percentage of people without advanced education earning more than 50K**.
* Finds the **minimum number of work hours per week**.
* Calculates the **percentage of people earning >50K among those who work minimum hours**.
* Determines the **country with the highest percentage of people earning more than 50K**.
* Finds the **most common occupation in India among people earning >50K**.

## Technologies Used

* Python
* Pandas

## Run the Project

```bash
python demographic_data_analyzer.py
```

The script will print all analysis results in the terminal.
